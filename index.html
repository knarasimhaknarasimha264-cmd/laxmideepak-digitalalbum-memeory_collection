📁 How to set up your album on GitHub

1
In your GitHub repo, create a folder called photos and upload all your images there
2
Create a file called photos.json in the root of the repo (same level as index.html) listing all your photo filenames: 
["photo1.jpg", "photo2.jpg", "IMG_001.jpg", "DSC_100.jpg"]
3
Go to Settings → Pages → Source: main branch → / (root) → click Save
4
Share the link with your father: https://yourusername.github.io/your-repo-name
💡 Easy way to make photos.json on Windows: Open the photos folder, press Shift + Right-click → "Open PowerShell here", then type:

Get-ChildItem -Name | ConvertTo-Json | Out-File ../photos.json

This automatically creates the photos.json file for you!
